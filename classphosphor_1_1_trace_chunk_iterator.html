Detailed Description

Const bi-directional iterator over the TraceBufferChunks in a TraceBuffer

Usage:

TraceChunkIterator chunk_iterator = buffer.chunk_start(); std::cout << *(chunk_iterator->start()) << std::endl;

for(const auto& chunk : buffer.chunks()) { for(const auto& event : chunk) { std::cout << event << std::endl; } }
